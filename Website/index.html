    • ในปี 2557 :: อันดับ 1 คือ กรุงเทพมหานคร อันดับ 2 คือ ภาคตะวันออกเฉียงเหนือ

    สรุปผลการวิเคราะห์และการคาดคะเนของผู้จัดทำ

    ....โดยจะเห็นได้ว่ามีภาคตะวันออกเฉียงเหนือ ที่ติดอันดับอยู่ด้วยกันถึง 3 ปี สมมติฐาน คนไปอาศัยอยู่ที่ภาคตะวันออกเฉียงเหนือเป็นจำนวนมากสาเหตุน่าจะมาจาก ภาคตะวันออกเฉียงเหนือเป็นภาคที่มีความสงบ และไม่ค่อยวุ่นวาย ซึ่งในยุคที่มีแต่ความวุ่นวายนี้ทำให้ มีผู้ต้องการหาความสงบให้ชีวิตของตน จึงย้ายไปอาศัยที่ภาคตะวันออกเฉียงเหนือเป็นส่วนใหญ่

  • 14.จำแนกผู้ย้ายถิ่นที่อยู่ก่อนย้ายและที่อยู่ปัจจุบัน

    คำอธิบายกราฟ

    เเกน x เป็นภาคต่างๆของประเทศไทย

    เเกน y เป็นอัตราส่วนที่ได้

    วิเคราะห์

    กราฟนี้เป็นกราฟแสดงจำนวนผู้ย้ายถิ่นที่อยู่ก่อนย้ายและที่อยู่ปัจจุบัน โดยผลที่ได้ทำให้ทราบได้ว่า 3 อันดับแรก คือ
    1. ย้ายมาจากภาคตะวันออกเฉียงเหนือไปอาศัยในกรุงเทพมหานคร
    2. ย้ายมาจากภาคกลางไปอาศัยอยู่ภาคตะวันออกเฉียงเหนือ
    3. ย้ายมาจากภาคกลางไปอาศัยอยู่ในกรุงเทพมหานคร

    สรุปผลการวิเคราะห์และการคาดคะเนของผู้จัดทำ

    ....โดยจะเห็นได้ว่าส่วนใหญ่จะมีภาคตะวันออกเฉียงเหนือ กรุงเทพมหานคร และภาคกลาง ที่มีจำนวนการย้าย เข้าและออกเป็นจำนวนมาก
    • สมมติฐานที่ 1 :: การที่คนในภาคตะวันออกเฉียงเหนือต้องการความเจริญจึงย้ายเข้ามาอยู่ในกรุงเทพมหานครและภาคกลางเพราะทั้ง สองส่วนนี้ที่เป็นภาคที่มีความเจริญมากที่สุด
    • สมมติฐานที่ 2 :: คนจากภาคกลางย้ายไปอยู่ภาคตะวันออกเฉียงเหนือน่าจะมาจากการต้องการความสงบไม่ ต้องการความวุ่นวายเหมือนในปัจจุบัน
    • สมมติฐานที่ 3 :: คนภาคกลางย้ายไปอยู่ในกรุงเทพมหานคร คาดว่าคงต้องความเจริญที่มากขึ้นจึงย้ายไปอาศัยอยู่ที่กรุงเทพมหานคร

  • 15.ร้อยละของผู้ย้ายถิ่นจำแนกตามสาเหตุของการย้ายถิ่่นฐาน

    คำอธิบายกราฟ

    เเกน x เป็นร้อยละทที่ได้

    เเกน y เป็นเหตุผลในการย้าย

    วิเคราะห์

    กราฟนี้เป็นกราฟวิเคราะห์ร้อยละของสาเหตุของการย้ายถิ่นฐานในแต่ละสาเหตุ โดยผลที่ได้จะทราบได้ว่า เหตุผลที่คนย้ายถิ่นฐาน 3 อันดับแรก คือ
    1. ติดตามคนในครอบครัว
    2. หางานทำ
    3. ทำกิจกรรมครอบครัว

    สรุปผลการวิเคราะห์และการคาดคะเนของผู้จัดทำ

    ....จะเห็นได้ว่า อันดับ 1 และ 3 นั้นเกี่ยวข้องกับเรื่องของครอบครัว ทำให้เราทราบได้ว่าครอบครัวมีส่วนสำคัญในการตัดสินใจที่จะย้ายถิ่นฐานเป็นอย่างมาก สมมติฐานที่ 1 เมื่อเรามีครอบครัวเราคงไม่อยากอาศัยอยู่คนเดียว ทำให้ถ้าเกิดมีใครคนหนึ่งที่ต้องย้ายที่อยู่อาศัยก็จะทำให้ คนอื่นๆ ในครอบครัวย้ายตามไปด้วย สมมติฐานที่ 2 เมื่อคนต้องการความก้าวหน้า ก็ต้องมีการหางานที่เหมาะสมสำหรับตนเอง และส่วนใหญ่งานที่เหมาะสมกับ บุคคลนั้นก็จะไม่อยู่ในใกล้กับที่ที่ตนเองอาศัยอยู่ ทำให้ต้องมีการย้ายที่อยู่อาศัยไปเพื่อให้ใกล้กับที่ทำงานของตน

  • 16.ร้อยละของจำนวนผู้ย้ายถิ่นจำแนกตามภาคที่อยู่

    คำอธิบายกราฟ

    เป็นกราฟวงกลมที่แสดงเป็นเปอร์เซนต์ของผู้ย้ายถิ่นฐานของประชากรเเต่ละภาคของประเทศไทย

    วิเคราะห์

    กราฟนี้จะแสดงอัตราส่วนของจำนวนผู้ที่ได้ย้ายเข้ามาอยู่ในภาคต่างๆ โดยมีอันดับการย้ายเข้าไปอาศัย ดังนี้
    1. ภาคตะวันออกเฉียงเหนือ
    2. ภาคกลาง
    3. ภาคเหนือ
    4. ภาคใต้
    5. กรุงเทพมหานคร

    สรุปผลการวิเคราะห์และการคาดคะเนของผู้จัดทำ

    ....โดยคาดว่าที่ ภาคตะวันออกเฉียงเหนือเป็นอันดับ 1 สาเหตุน่าจะมาจากเป็นภาคที่มี ความสงบสุข ทำให้มีผู้ต้องการย้ายเข้าไปอยู่อาศัยเป็นจำนวนมาก สำหรับเหตุผลที่ภาคกลางได้อันดับที่ 2 ไป สาเหตุน่าจะมาจากปัญหาความวุ่นวายในภาคกลาง แต่ถึงจะมีปัญหาต่างๆ แต่ก็ถือเป็นภาคที่มีความเจริญค่อนข้างสูง ทำให้มีผู้ย้ายเข้าไปเป็นจำนวนมาก ในส่วนสาเหตุที่ภาคเหนือ ได้อันดับที่ 3 นั้น อาจเป็นเพราะความหนาวเย็นของภาคเหนือที่หนาวมากเกินไป ทำให้มีคนที่ย้ายไปอยู่ที่ภาคนั้นเป็นอันดับ 3 สำหรับภาคใต้ ที่เป็นอันดับที่ 4 สาเหตุน่าจะมาจากปัญหาความไม่สงบในภาคนี้ ทำให้มีคนที่ ตัดสินใจย้ายถิ่นที่อยู่ไปอาศัยอยู่ในภาคใต้นี้เป็นจำนวนนวนน้อย และสำหรับกรุงเทพมหานครที่มีอันดับสุดท้าย อาจเป็นเพราะอัตราส่วนของพื้นที่ที่น้อยกว่าพื้นที่ของ ภาคอื่นๆ และความวุ่นวายที่เกิดขึ้นในกรุงเทพทำให้มีจำนวนผู้เข้าไปอาศัยเพียงเท่านี้
